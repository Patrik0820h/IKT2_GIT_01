# Csapatmunka Beadandó

## Feladat leírás
Ez a repository az IKT tantárgy Git-es feladatához készült.  
A projekt célja a Git verziókezelés, valamint az alap webes technológiák rövid bemutatása és gyakorlása.

## Felhasznált technológiák
- GIT
- HTML
- CSS
- JavaScript

## Kifejtésük

- **GIT**: Verziókezelő rendszer, amely segít nyomon követni a projektben végzett változtatásokat.
Lehetővé teszi, hogy több fejlesztő egyszerre dolgozzon ugyanazon a kódon.
Segítségével vissza lehet térni korábbi állapotokra, ha valami elromlik.
