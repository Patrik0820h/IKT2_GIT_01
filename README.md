# Csapatmunka Beadandó

## Feladat leírás
Ez a repository az IKT tantárgy Git-es feladatához készült.  
A projekt célja a Git verziókezelés, valamint az alap webes technológiák rövid bemutatása és gyakorlása.

## Felhasznált technológiák
- GIT
- HTML
- CSS
- JavaScript

## Kifejtésük

- **GIT**: Verziókezelő rendszer, amely segít nyomon követni a projektben végzett változtatásokat.
Lehetővé teszi, hogy több fejlesztő egyszerre dolgozzon ugyanazon a kódon.
Segítségével vissza lehet térni korábbi állapotokra, ha valami elromlik.

| Rövid név | Teljes név              | Típus            | Cél |
|-----------|-------------------------|------------------|-----|
| GIT       | Git Version Control     | Verziókezelő     | Kódváltozások kezelése |
| ⭐⭐⭐⭐☆    |                         |                  |     |

- **HTML**: A weboldalak szerkezetét meghatározó jelölőnyelv.
Segítségével lehet felépíteni az oldal tartalmát, például címsorokat és bekezdéseket.
Minden weboldal alapja, böngészőben közvetlenül értelmezhető.

| Rövid név | Teljes név              | Típus        | Cél |
|-----------|-------------------------|--------------|-----|
| HTML      | HyperText Markup Language | Jelölőnyelv | Weboldal szerkezete |
| ⭐⭐⭐⭐⭐    |                         |              |     |

- **CSS**: A weboldalak kinézetéért felelős stílusleíró nyelv.
Segítségével beállíthatók a színek, betűtípusok és az elrendezés.
HTML nélkül nincs értelme, együtt adják a weboldal alapját.

| Rövid név | Teljes név              | Típus            | Cél |
|-----------|-------------------------|------------------|-----|
| CSS       | Cascading Style Sheets  | Stílusleíró nyelv | Megjelenés formázása |
| ⭐⭐⭐⭐☆    |                         |                  |     |

- **JavaScript**: Programozási nyelv, amely interaktívvá teszi a weboldalakat.
Lehetővé teszi a felhasználói események kezelését, például kattintásokat.
Dinamikus tartalmak és logika megvalósítására szolgál.

| Rövid név | Teljes név   | Típus              | Cél |
|-----------|--------------|--------------------|-----|
| JS        | JavaScript   | Programozási nyelv | Interaktivitás |
| ⭐⭐⭐☆☆    |              |                    |     |
