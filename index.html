Git parancsok

git add name

Nem nyomon követett fájl vagy nem előkészített módosítások hozzáadása:

git commit -m ""

Commitot készítesz

git status

Ellenőrizze mit adott hozzá:

git pull

távoli adattárból tartalom lekérésére és letöltésére, valamint a helyi adattár azonnali frissítésére szolgál, hogy az megfeleljen az adott tartalomnak.
